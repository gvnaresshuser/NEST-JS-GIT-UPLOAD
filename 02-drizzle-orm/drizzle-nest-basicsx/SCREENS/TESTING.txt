# 🧪 1. CREATE User

### ✅ URL

```bash
POST http://localhost:3000/users
```

### ✅ Body (JSON)

```json
{
  "name": "Naresh",
  "email": "naresh@test.com",
  "password": "123456",
  "profession":"Engineer",
  "address":"Plot:186/12 MVP COLONY"
}
```
RESPONSE:
---------
{
  "id": 1,
  "name": "Naresh",
  "email": "naresh@test.com",
  "password": "123456",
  "address": "Plot:186/12 MVP COLONY",
  "profession": "Engineer"
}
---------------
{
  "name": "John",
  "email": "john@test.com",
  "password": "654321",
  "profession":"Assistant",
  "address":"Plot:12/186 ABC COLONY"
}

### 🧪 cURL

```bash
curl -X POST http://localhost:3000/users \
-H "Content-Type: application/json" \
-d '{"name":"Naresh","email":"naresh@test.com","password":"123456"}'
```

# ✏️ 4. UPDATE User

### ✅ URL

```bash
PATCH http://localhost:3000/users/1
```

### ✅ Body

```json
{
  "name": "Naresh Updated Name"
}
```
RESPONSE:
--------
{
  "name": "Updated Name",
  "email": "test_naresh@test.com",
  "password": "111111"
}

### 🧪 cURL

```bash
curl -X PATCH http://localhost:3000/users/1 \
-H "Content-Type: application/json" \
-d '{"name":"Updated Name"}'
```

---

# ❌ 5. DELETE User

### ✅ URL

```bash
DELETE http://localhost:3000/users/2
```
RESPONSE:
--------
{
  "message": "User deleted successfully",
  "data": {
    "id": 3,
    "name": "Updated Name",
    "email": "naresh@test.com",
    "password": "123456"
  }
}

### 🧪 cURL

```bash
curl -X DELETE http://localhost:3000/users/1
```

# ⚠️ Validation Test Cases

### ❌ Missing name

```json
{
  "email": "test@test.com",
  "password": "123456"
}
```

👉 Expected:

```json
{
  "message": ["Name is required"]
}
```

---

### ❌ Invalid email

```json
{
  "name": "Test",
  "email": "wrong-email",
  "password": "123456"
}
```

---

### ❌ Short password

```json
{
  "name": "Test",
```
