```bash

```

---

## ✅ `db.ts`

```ts
import 'dotenv/config';
import { drizzle } from 'drizzle-orm/node-postgres';
import { Pool } from 'pg';

const pool = new Pool({
  connectionString: process.env.DATABASE_URL,
});

export const db = drizzle(pool);
```

---

## ✅ `users.schema.ts`

```ts
import { pgTable, serial, text } from 'drizzle-orm/pg-core';

export const users = pgTable('nestjs_users_new', {
  id: serial('id').primaryKey(),
  name: text('name'),
  email: text('email'),
  password: text('password'),
});
```

# ⚙️ 5. Drizzle Config

## 📁 Root: `drizzle.config.ts`

  out: './drizzle',

# ⚠️ 12. IMPORTANT: dotenv vs ConfigModule

This is where things get interesting 👇

---

## 🟡 Option 1: `dotenv` (what you’re using now)

```ts
import 'dotenv/config';
```

✔ Simple
✔ Works immediately
❌ Not NestJS-native
❌ Harder to scale

---

## 🟢 Option 2: NestJS ConfigModule (Recommended)

Use @nestjs/config

---

# 🔥 Setup ConfigModule

## Step 1: Install

```bash
pnpm add @nestjs/config
```

---

## Step 2: Update `app.module.ts`

```ts
import { ConfigModule } from '@nestjs/config';

@Module({
  imports: [
    ConfigModule.forRoot({
      isGlobal: true,
    }),
    UsersModule,
  ],
})
export class AppModule {}
```

---

## Step 3: Use in `db.ts`

```ts
import { ConfigService } from '@nestjs/config';
```

👉 Better approach: inject config (advanced)

But for now:

```ts
import 'dotenv/config';
```

OR later refactor to:

```ts
const configService = new ConfigService();
configService.get('DATABASE_URL');
```

---

# 🧠 dotenv vs ConfigModule

| Feature            | dotenv | ConfigModule |
| ------------------ | ------ | ------------ |
| NestJS integration | ❌      | ✅            |
| Type-safe          | ❌      | ✅            |
| Scalable           | ❌      | ✅            |
| Quick setup        | ✅      | ✅            |

---

# 🏆 Best Practice (Production)

👉 Use BOTH:

* `dotenv` → for Drizzle CLI (`drizzle.config.ts`)
* `ConfigModule` → for NestJS app

# ⚠️ 2. Important Fix (SSL for Render PostgreSQL)

Since you're using **Render PostgreSQL**, you MUST enable SSL.

Update your `db.ts` 👇

```ts id="9cqlr2"
import 'dotenv/config';
import { drizzle } from 'drizzle-orm/node-postgres';
import { Pool } from 'pg';

const pool = new Pool({
  connectionString: process.env.DATABASE_URL,
  ssl: {
    rejectUnauthorized: false, // IMPORTANT for Render
  },
});

export const db = drizzle(pool);
```
