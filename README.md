# 🚀 How to Run Any Project

## 1️⃣ Clone repo

```bash
git clone https://github.com/YOUR_USERNAME/NEST-JS-GIT-UPLOAD.git
cd NEST-JS-GIT-UPLOAD
```

---

## 2️⃣ Go to project folder

```bash
cd 02-drizzle-orm/drizzle-nest-basics
```

---

## 3️⃣ Install dependencies

```bash
pnpm install
```

---

## 4️⃣ Setup environment variables

```bash
cp .env.example .env
```

---

## 5️⃣ Run project

### 👉 NestJS

```bash
pnpm start:dev
```

### 👉 React (Vite)

```bash
npm run dev
```

---

# ⚠️ Important Notes

* Each project is **independent**
* Run commands **inside project folder**
* Do NOT run from root
* Setup `.env` before running
